## 🗂️ Current Courses

### 1. [Python for Data Science](./PYTHON%20FOR%20DATA%20SCIENCE/)
A comprehensive introduction to Python fundamentals, laying the groundwork for data analysis and programming logic. The course is structured day-by-day:
* **Day 1 & 2**: Intro to OOP, Jupyter Notebooks, variables, strings, and numerical operations.
    * *Mini-Project*: Currency Exchange Calculator.
* **Day 3**: String manipulation, indexing, formatting, and handling user inputs.
    * *Mini-Project*: Text Analyzer.
* **Day 4**: Core data structures (Lists, Tuples, Dictionaries), Booleans, and Control Flow (If/Elif/Else).
    * *Mini-Project*: Interactive Contact Book.
* **Day 5**: Loops (For/While) and custom Functions.
    * *Mini-Project*: Multiplication Table Generator.

### 2. [Total SQL](./SQL_TOTAL/)
Practical exercises and scripts for database design and management.
* **Netflix Database Project**:
    * `Netflix_schema.sql`: DDL scripts creating the relational schema for a streaming platform (Series, Episodes, Actors, Actuaciones).
    * `Netflix_data.sql`: DML scripts populating the database with realistic data from popular shows like *Breaking Bad*, *Stranger Things*, *The Witcher*, *The Office*, and more.
